# Exercícios — Programação I ECO 2026

Esta pasta contém as entregas de exercícios de cada aula.

## Organização

Cada aula terá uma subpasta no formato `aula-NN/`. Dentro de cada aula, cada aluno deve criar uma pasta com o seu próprio nome.

```
exercicios/
├── aula-01/
│   ├── renan-ramlov/
│   └── vinicius-alves/
├── aula-02/
│   ├── renan-ramlov/
│   └── ...
└── ...
```

## Como entregar

1. Vá até a pasta da aula: `exercicios/aula-NN/`
2. Crie uma pasta com seu nome (ex.: `renan-ramlov`)
3. Coloque seus arquivos `Java` ou 'C' nela
4. Faça `git add`, `git commit` e `git push`

> Consulte o guia completo na página da disciplina.
